Inherited from Widget.hiddenProperty

  • Defined in /home/travis/build/phosphorjs/phosphor-menus/node_modules/phosphor-widget/lib/widget.d.ts:125

A property descriptor which controls the hidden state of a widget.

Notes

This controls whether a widget is explicitly hidden.

Hiding a widget will cause the widget and all of its descendants to become not-visible.

This will toggle the presence of 'p-mod-hidden' on a widget. It will also dispatch 'after-show' and 'before-hide' messages as appropriate.

The default value is false.

See also: hidden, isVisible

Static itemsProperty

itemsProperty: Property<MenuBase, Array<MenuItem>>
  • Defined in menubase.ts:47

The property descriptor for the array of menu items.

Notes

This property creates a frozen shallow copy of the assigned items array. This means that the menu items can only be changed in bulk and that in-place modifications to the array are not allowed.
