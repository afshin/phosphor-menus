Static commandProperty

commandProperty: Property<MenuItem, string>
  • Defined in menuitem.ts:221

The property descriptor for the item command id.

If a command id is provided, the handler for the menu item will be ignored, and the Menu widget which consumes the menu item will request that the given command be executed instead.

See also: command
