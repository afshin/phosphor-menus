trigger

  • trigger(): void
    • Defined in menuitem.ts:470

    Trigger the menu item.

    Notes

    This will emit the triggered signal. If the item is checkable, it will first be toggled.

    This will typically be called by the widget which renders the menu item in response to a user mouse click.

    Returns void
